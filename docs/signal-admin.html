Manage what your weekly scan searches for — no GitHub UI required. Changes here save straight to your repo, and the next scan (scheduled or manual) picks them up automatically.

Connect

Paste a GitHub personal access token with write access to signal-scout2. It's stored only in this browser's local storage and sent directly to GitHub's API — never to any other server, including ours.

How to create a token GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Set repository access to only signal-scout2, and under Permissions grant Contents: Read and write. Nothing else is needed.
